# Fast-Experiment-tracking-Demo
Fast experiment tracking with DVCLive . Satellite Orbit prediction challenge.

To reproduce the Demo 

## Download the data 

This command will create a
   folder inside the project called satellite-data containing the dataset.

```
dvc get https://github.com/iterative/dataset-registry \
          workshop/satellite-data/
```

## Initialize project

Initialize DVC. You can go to the extension tab and click the button

## Show results

Run the notebook and go to Show experiments tab
Repeat until you can find the suitable experiment
